# Warehouse-Management-System-WMS-
The Enterprise Warehouse Management System (WMS) is a cloud-based inventory and logistics management platform developed using Spring Boot, Java, and PostgreSQL to automate warehouse operations for modern e-commerce and logistics companies.

                   Technologies Used

     Technology	                           Purpose
        Java	                         Backend programming
    Spring Boot                      	Application framework
      Spring Data JPA                 	Database operations
     PostgreSQL	                            Database
      Spring Security	               Authentication & authorization
      JWT                             	Secure API authentication
